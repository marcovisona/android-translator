📦 INSTALLATION
────────────────────────────────────────────────────────────────────
# With Poetry (recommended)
poetry install
poetry run android-translator --help

# Alternative: with pip
pip install --user pandas openpyxl
poetry run android-translator --help

📤 EXPORT (Android → Excel)
────────────────────────────────────────────────────────────────────
poetry run android-translator strings export \
    /path/to/project/module/src/main \
    output.xlsx

Options:
  --default-language LANG    Default language (default: en)

📥 IMPORT (Excel → Android)
────────────────────────────────────────────────────────────────────
poetry run android-translator strings import \
    output.xlsx \
    /path/to/project/module/src/main

Options:
  --default-language LANG    Default language (default: en)

📤 EXPORT (HTML → Excel)
────────────────────────────────────────────────────────────────────
poetry run android-translator html export \
    /path/to/project/module/src/main/assets/html \
    output_dir

Options:
  --keep-html-tags    Keep HTML tags (default: remove)

📥 IMPORT (Excel → HTML)
────────────────────────────────────────────────────────────────────
poetry run android-translator html import \
    output_dir \
    /path/to/project/module/src/main/assets/html
